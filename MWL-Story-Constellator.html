◈ Insert Proper Name Cartouche

PNG files in ./proper-names/ folder. The name is case-sensitive. Set the glyph size in Constellator to match your PNC generator output (64, 96, 128, or 512 px).

⑤ Insert Numeral Glyph

MWL numerals 2–8 have dedicated glyphs in ./MWL-glyphs/. For values ≥9, use the binary rail numeral generator tool to create the SVG, place it in the glyphs folder, and reference it here by filename.
